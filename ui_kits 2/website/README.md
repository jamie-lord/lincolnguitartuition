# Website UI Kit — Lincoln Guitar Tuition

A modern, dark-and-moody redesign of `lincolnguitartuition.uk`. The original IA is preserved (About / Contact / FAQs / Backing tracks / TRAP / Testimonials / Videos / Session backing) — only the visual design changes.

## Files

- `index.html` — interactive single-page demo. Click nav to jump between sections.
- `Nav.jsx` — sticky top nav, frosts on scroll
- `Hero.jsx` — full-bleed hero with eyebrow + display headline + dual CTA
- `LessonCard.jsx` — lesson tier card (in-person / online / sessions)
- `BackingTrackPlayer.jsx` — track row with mono timestamp + amber play button
- `TestimonialBlock.jsx` — pull-quote + attribution
- `ContactBlock.jsx` — phone / email / studio address
- `Footer.jsx` — wordmark + nav + contact
- `BioSection.jsx` — about Dan layout (image left, prose right)

## Notes

- Photos: SVG placeholders are used. Drop real photos into `../../assets/photos/` and replace the references.
- This is a static visual kit — buttons go nowhere, but hover/focus states all work.
